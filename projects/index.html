Completed Projects

Differential Drive Donkey Car
  • Built a differential drive Donkey Car, a drivereless small scale vehicle.
  • A TensorFlow based Autopilot was trained on nearly 10,000 images of a track.
  • The Autopilot was then used to drive the vehicle autonomously. Check it out here!
4 Degrees of Freedom Gesture Controlled Robotic Arm
  • 3D printed robotic arm controlled by human gestures through a Microsoft Kinect Sensor.
  • Right arm gestures were mimicked by the Arm, whereas left arm movements were used for directional orientation of the robot.
  • The code was written in primarily Arduino and Processing, paired with OpenNI library for collection of skeletal data.
Fracture-Detection System
  • Built a fracture detection system to identify fractures in manufactured components, and predictions were made about the same.
  • The system employed OpenCV through which fractures were identified, after being subjected to different filters.
  • A TensorFlow model, trained on a dataset of normal and fractured gear images was used as an example to make predictions and test the system.
Ultrasonic Based Obstacle Avoider
  • Built an Arduino based mobile robot, capable of avoiding obstacles.
  • Bot comprised of three ultrasonic sensors oriented in three different directions which actively detect obstacles around them.
  • The project was a great starting point to learn the basics of Arduino programming.
Line Follower Bot
  • Built a line follower bot that followed a black stripe overlaid on a white track.
  • The Bot has an Arduino Uno onboard, which co-ordinates the movements of the bot with the sensor readings, making it follow a black line using three infrared sensors.
  • This project was one of the first projects built once inducted into the robotics club.
