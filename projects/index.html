Projects

pyResearchInsights: An open-source tool for scientific text analysis [Code]
  • We developed pyResearchInsights an end-to-end, open-source, automated content analysis tool that Scrapes abstracts, Cleans them up, Analyses temporal frequency of keywords and Visualizes themes of discussions.
  • Collaboration with Vijay Ramesh from the Department of Ecology, Evolution and Environmental Biology, Columbia University, and Anand MO from the Nature Conservation Foundation.
  • Package built using Pandas, BeautifulSoup, gensim and spaCy.
IUCN Red List species threats and stressess database [Code]
  • Generated a database on species threats and stresses by scrapping the IUCN Red List.
  • Collaboration with Uttara Mendiratta and Anand MO from the Nature Conservation Foundation.
  • Utilized Selenium Web Driver to automate the scrapping procedure and BeautifulSoup to scrape the HTML contents. Pandas was used to build dataframes and interface with a local .CSV file.
Visualization Tools for Operation Research [Code]
  • Building a tool to visualize the critical path between start and end nodes in a network of activities.
  • Currently working towards determining ideal path and shortest paths using graph theoretic approaches.
  • Code is written primarily in Python, and uses the Graphviz library to build the network.
Fracture-Detection System [Code] [Paper] [arXiv]
  • Built a fracture detection system to identify fractures and fatiguing in manufactured components, for Design of Machine Elements Course - II (12ME62).
  • The system employed OpenCV through which fractures were identified, after being subjected to different filters.
  • The TensorFlow model, based on ImageNet, retrained on a dataset of normal and fractured gear images, was used as an example to make predictions about wearability and test the system.
Biomimetic Fish [Code]
  • Built a predatory fish, as a part of Summer Research Internship at the Theoretical Ecology & Evolution Laboratory
  • The Fish will be used to research collective behavior and aggregation patterns in Serpae tetra species of fish.
  • Worked under the guidance of Professor Vishwesha Guttal from Center for Ecological Sciences, Indian Institute of Science.
Detection of Algal Colonies [Code]
  • Working to develop a image processing system for the identification of different algal colonies in a dataset of coral-reef images.
  • Working in collaboration with Sumithra Sankaran from the Theoretical Ecology & Evolution Laboratory.
  • The system is based on OpenCV, carrying out different image processing techniques to efficiently identify different algal colonies in the image.
Differential Drive Donkey Car [Code]
  • Built a small-scale differential drive autonomous car, powered by a TensorFlow based Autopilot, trained on nearly 10,000 images of a track.
  • The Autopilot was used to drive the vehicle autonomously around the track. Check it out here!
  • Research undertaken at the Center for System Design at the National Institute of Technology Karnataka, jointly advised by Professor Gangadharan KV and Professor Pruthviraj Umesh.
Gesture Controlled Arm [Code]
  • Built and designed a 3D printed arm, controlled by human gestures using a Microsoft Kinect 360.
  • Open Natural Interactions Library for Processing was used to detect user's pose and joint angles.
  • Detected angles were translated to joint movements of the Arm by an Arduino Uno.
Ultrasonic Obstacle Avoider
  • Built an Arduino based bot, capable of avoiding obstacles.
  • Bot comprised of three ultrasonic sensors oriented in three different directions which actively detect obstacles around them.
  • The project was also a great starting point to learn the basics of Arduino programming.
Line Follower Bot
  • Built a line follower bot that followed a black stripe overlaid on a white track.
  • The Bot has an Arduino Uno onboard, which co-ordinates the movements of the bot with the sensor readings, making it follow a black line using three infrared sensors.
  • The project was a great starting point to learn the basics of Arduino programming.
